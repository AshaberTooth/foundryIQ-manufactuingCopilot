# Alarm Code Reference Guide

## Alarm E-217

Alarm E-217 indicates conveyor stoppage caused by one or more of the following:

- Downstream obstruction
- Photo-eye sensor misalignment
- Motor overload relay trip
- Excessive belt tension
- Intermittent sensor bracket vibration

Recommended diagnostic order:

1. Perform lockout/tagout before physical inspection.
2. Inspect photo-eye sensor alignment.
3. Check sensor bracket vibration.
4. Inspect conveyor path for obstruction.
5. Check belt tension.
6. Review motor temperature and overload relay status.
